# Django To-Do App

A simple, authenticated to-do list application built with Django.  
Users can register, log in, and manage their own tasks with features like adding, toggling completion, deleting, filtering, and viewing statistics.

## Features

- User authentication (registration, login, logout)
- Add, delete, and toggle task completion
- Filter tasks by All, Active, or Completed
- Display total, active, and completed task counts
- Each user sees only their own tasks

## Tech Stack

- Python 3.x
- Django 6.0
- SQLite (default)
- HTML/CSS (custom styling)
